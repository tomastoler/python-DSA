# Python Certificates App
Repositorio para practicar el uso de github con certificados ssh, python, entornos virtuales y docker.

```bash
git clone <repo>
```

## Iniciar el proyecto
```bash
docker compose up --build -d
```

## Routes
```yml
app
  > auth
    > login
    > sign-up
    > logout
    > certify
  > notes
    > certify
```
